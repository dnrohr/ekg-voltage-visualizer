# V2 Guided Lessons

The guided lesson panel is a non-blocking overlay on the live simulator. Selecting a lesson updates the current time, selected lead, selected region, and learner mode, but users can continue changing any control afterward.

## Coverage

1. Lead as directional view.
2. Limb leads.
3. Precordial leads and R-wave progression concepts.
4. Normal ventricular depolarization.
5. Mechanical delay.
6. Reconstructing heart activity from leads.

Each lesson uses existing simulation state rather than static screenshots. Quiz feedback is immediate, model-scoped, and non-clinical.

## Safety

Quiz wording must avoid diagnostic claims. Feedback should teach the relationship among vector direction, surface regions, leads, and mechanical timing inside this synthetic model.
