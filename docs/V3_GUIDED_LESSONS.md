# V3 Guided Lessons

V3 lessons supersede the original V2 lesson list while preserving the same non-blocking workflow. Selecting a lesson updates:

- scenario and comparison scenario
- cardiac-cycle time
- selected lead and selected mesh region
- learner mode and layer preset
- 3D camera preset and anatomy view mode

The lesson card also names the V3 mesh focus so learners know which visual layer to inspect before answering the quiz.

## V4 Anatomy Addendum

V4 keeps the same non-blocking lesson workflow and updates lesson copy where anatomy improves the learning objective:

- Normal QRS propagation references the hybrid anatomical overlay and the septal/apical anchor sequence.
- Limb lead axes points learners to anatomical markers plus contributor halos.
- Precordial lead views connects LV lateral-wall selection with the apex-forming LV in the orientation sketch.
- Bundle-branch delay and ectopic-focus lessons use cutaway/chamber views while preserving heart-timing-before-ECG framing.
- Repolarization references blue recovery rings near anatomical anchors while repeating that recovery timing is authored.

The lesson panel remains non-diagnostic. NIH anatomy is described as a visual reference; wavefront timing, lead voltages, and quiz feedback come from the authored teaching model.

## Lesson Coverage

1. Normal QRS propagation: follows the V3 mesh depolarization band and current isochrone contour through septal/apical activation.
2. Limb lead axes: connects aVR and limb-lead polarity to the selected lead probe and contributor halos.
3. Precordial lead views: uses LV lateral-wall selection to compare best-seen and opposite precordial views.
4. Bundle-branch delay comparison: uses cutaway anatomy plus the comparison viewer to emphasize altered heart timing before QRS widening.
5. Ectopic ventricular focus: uses chamber view to inspect an early RV free-wall source and its lead effects.
6. Repolarization and T wave: separates blue recovery-band behavior from depolarization and reinforces that recovery is not the QRS wave running backward.
7. Normal vs abnormal comparison: uses the V3 comparison viewer heart-first, then examines voltage delta.

## Quiz Rules

Quiz prompts must avoid diagnostic claims. Feedback should reinforce the model-scoped relationship among mesh wavefront timing, selected lead definitions, region metadata, scenario comparison, and generated ECG traces.
