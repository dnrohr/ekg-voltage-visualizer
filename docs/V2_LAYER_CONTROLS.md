# V2 Layer Controls

V2 uses learner modes as saved layer presets. The controls preserve current time, selected lead, selected region, scenario, and comparison; switching modes only changes layer visibility.

## Learner Modes

| Mode | Purpose |
|---|---|
| Lead to heart | Beginner default for selecting a lead and seeing what the heart is doing. |
| Heart to leads | Starts from a selected surface region and emphasizes matching ECG traces. |
| Advanced | Enables the full electrical, mechanical, and ECG overlay set. |

## Layer Groups

Electrical layers:

- Wavefront
- Contours
- State map
- Net vector
- Lead probe

Mechanical layers:

- Contraction
- Chamber volume
- Valves
- Flow
- Phase labels

ECG layers:

- 12-lead grid
- Enlarged lead
- Labels
- Projection marker
- Region highlights

Layer choices are saved in the existing view preset alongside selected lead, selected region, scenarios, contrast, and motion settings.
